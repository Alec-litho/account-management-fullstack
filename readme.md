# account-management-fullstack-test

# Frontend
## navigate to backend
```
cd frontend
```
## Project setup
```
npm install
```
### Compiles and hot-reloads for development
```
npm run serve
```
# Backend
## navigate to backend
```
cd backend
```
## Project setup
```
npm install
```
## run server
```
npm run start:dev
```
# Note

Server automatically seeds when you run it
